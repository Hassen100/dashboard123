## 📋 Projet Angular 17

Application d'authentification moderne avec :
- **Login / Sign Up** avec Reactive Forms
- **Design responsive** et animations fluides
- **Validation complète** des formulaires
- **Style moderne** avec glassmorphism

## 🛠️ Technologies

- **Angular 17** avec Ivy
- **TypeScript** pour la typé statique
- **CSS3** avec animations
- **Reactive Forms** pour la validation
