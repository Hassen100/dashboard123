# SEO Pulse Board

Dashboard SEO interactif avec graphiques en temps réel et recommandations IA.

## 🚀 Déploiement

Ce projet utilise GitHub Pages pour le déploiement automatique.

### Instructions:

1. **Activer GitHub Pages:**
   - Allez dans Settings > Pages
   - Source: "Deploy from a branch"
   - Branch: "main"
   - Folder: "/(root)"

2. **Déclencher le workflow:**
   - Allez dans l'onglet "Actions"
   - Cliquez sur "Deploy Angular to GitHub Pages"
   - Cliquez sur "Run workflow"

3. **Visiter le site:**
   - https://hassen100.github.io/Dashboard/

## 📊 Fonctionnalités

- KPIs en temps réel
- Graphiques interactifs (Chart.js)
- Filtres avancés
- Recommandations IA
- Design responsive

## 🛠️ Stack

- Angular 17
- Chart.js 4.4
- GitHub Actions
- GitHub Pages
